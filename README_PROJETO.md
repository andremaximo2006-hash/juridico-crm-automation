# 🚀 CRM Jurídico — Sistema IA de Atendimento WhatsApp

**Status:** 🟢 **LIVE EM PRODUÇÃO** | **Domínio:** https://crm.gabriellenunes.com.br | **Versão:** 1.0.0

---

## 📌 O Que É Este Projeto?

Um **CRM completo para escritórios jurídicos** com **IA de Atendimento automática** que:
- ✅ Recebe leads do WhatsApp (Z-API, Meta Business, ManyChat)
- ✅ Atende automaticamente com Claude AI
- ✅ Permite pausar e transferir para atendente humano
- ✅ Gerencia fila de atendimento
- ✅ Todos os roteiros são **configuráveis sem código**

---

## ✨ 3 Páginas IA Principais

### 📋 Roteiros (`/ia/roteiros`)
Editor de system prompts com versionamento automático. Customize a IA para cada especialidade jurídica sem tocar em código.

### 💬 Conversas (`/ia/conversas`)
Histórico completo de todas as conversas com clientes. Veja quem falou com a IA, e quando ela transferiu para atendente humano.

### 👥 Atendimento Humano (`/ia/atendimento-humano`)
Fila de tickets para seus atendentes. Priorização automática, atribuição de tarefas, e acompanhamento de status.

## 🏗️ Stack Técnico

```
Frontend:    Next.js 16 + TypeScript + Tailwind
Backend:     FastAPI (Python) + Claude AI
Database:    PostgreSQL + Prisma ORM
Produção:    VPS Ubuntu + Nginx + PM2
Integrações: Z-API, Meta, ManyChat, Asaas
```
