# ⚡ Quick Start - 5 Minutos

## 🎯 Seu Objetivo Hoje
**Qualificar e vender o máximo de leads com menos esforço**

---

## 🔑 Step 1: Login (30 segundos)

```
https://crm.gabriellenunes.com.br
Email: andre.maximo@gabriellenunes.com.br
Senha: Teste@123
```

---

## 🚀 Step 2: Abra o Hub (30 segundos)

**Navegação:**
```
Sidebar esquerdo 
  → IA Atendimento ▼
    → 🚀 IA Hub
```

Você verá:
- ✅ 12 módulos (Cards coloridos)
- ✅ 4 KPIs (Números no rodapé)

---

## 🎯 Step 3: Filtre seus Leads (2 minutos)

```
Menu → IA Atendimento → Filtros
```

**Selecione:**
- Tipo de Benefício: INSS (ou seu preferido)
- Score Mínimo: 70%
- Período: Últimos 7 dias

**Clique:** "Aplicar Filtros"

Resultado: Lista de leads prontos pra vender! 📋

---

## 📞 Step 4: Ligue para o Primeiro Lead (1 minuto)

Antes de ligar:
1. Clique no lead
2. Leia a conversa com Marta
3. Veja o score (deve estar > 70%)
4. **Anote 3 pontos principais**

Depois ligue: Já sabe tudo sobre ele! 

---

## ✅ Step 5: Feche a Venda (30 segundos)

Depois de vender:
1. Volte para o lead
2. Marque status: "Convertido"
3. Adicione nota: "Fechado! R$ XXXX"

Pronto! ✨

---

## 📊 Como Saber se Está Funcionando

**Todos os dias:**
```
Menu → IA Atendimento → Dashboard
Veja: Taxa de Conversão
Se está subindo = está funcionando!
```
