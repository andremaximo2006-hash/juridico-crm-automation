## 🎓 Roteiro de Treinamento Recomendado

### Semana 1

**Dia 1 - Onboarding Individual (30 min)**
- [ ] Vendedor lê QUICK_START_VENDAS.md (5 min)
- [ ] Você observa primeira navegação (10 min)
- [ ] Vendedor faz primeiro filtro (5 min)

- [ ] Vendedor usa CRM sozinho

- [ ] Vendedor lê documentação conforme precisa

**Dia 4 - Treinamento em Grupo (60 min)**
- [ ] Você conduz [SCRIPT_TREINAMENTO_AO_VIVO.md](./SCRIPT_TREINAMENTO_AO_VIVO.md) com TODA equipe
- [ ] Todos fazem filtro ao mesmo tempo
- [ ] Q&A ao final

**Dia 5 - Consolidação**
- [ ] Cada vendedor faz primeira venda
- [ ] Celebrem junto! 🎉
