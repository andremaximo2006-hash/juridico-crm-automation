## 💡 Como Usar

### Para Customizar a IA:
1. Vá para `/ia/roteiros`
2. Selecione uma especialidade
3. Edite o system prompt
4. Clique "Salvar" (versiona automaticamente)
5. Próxima conversa usa novo prompt ✅

### Para Ver Histórico:
1. Vá para `/ia/conversas`
2. Busque por cliente ou telefone
3. Veja todo o histórico (IA + Cliente)
4. Veja quando transferiu para atendente

### Para Gerenciar Tickets:
1. Vá para `/ia/atendimento-humano`
2. Veja fila de tickets
3. Atribua a um atendente
4. Acompanhe o status
