## 🎯 Visão Geral

O **CRM Jurídico** é uma plataforma inteligente que combina:
- **IA Conversacional** para qualificação automática de leads
- **4 Especialistas IA** em benefícios (INSS, BPC, Maternidade, Acidente)

✅ **Aumento de conversão** com dados estruturados  
✅ **Rastreamento completo** de todas as conversas

## 🔄 Fluxo de Atendimento

### Passo a Passo Completo

#### **Fase 1: Lead Chega (Você Não Faz Nada)**
- Cliente entra em contato via WhatsApp
- Sistema detecta automaticamente
- Marta inicia conversa

#### **Fase 2: Qualificação Automática (Marta)**
**Tempo:** 5-10 minutos
- Marta coleta: nome, idade, tipo benefício, documentos
- Sistema gera score preliminar
- Dados ficam estruturados no banco

#### **Fase 3: Roteamento (Automático)**
**Tempo:** Instantâneo
- Orquestrador analisa dados
- Roteia para especialista apropriado
- Se score baixo, Marta continua coletando

#### **Fase 4: Especialista Resolve (IA)**
**Tempo:** 10-15 minutos
- Especialista aprofunda conversa
- Valida viabilidade
- Gera parecer técnico
- Score final sobe para 70-100%

#### **Fase 5: Export para Vendedor (VOCÊ)**
**Seu trabalho começa aqui:**
- Recebe lead qualificado no Dashboard
- Acessa histórico completo da conversa
- Faz contato pessoal via telefone
- Negocia e fecha o caso

## 📊 Métricas e Relatórios

### KPIs Principais

| KPI | O Que Significa | Objetivo |
|---|---|---|
| **Conversas Totais** | Total de leads que iniciaram conversa | Quanto maior, melhor |
| **Especialistas Ativos** | Quantos especialistas estão disponíveis | Deve ser 4 |
| **Score Médio** | Média de viabilidade de todos os leads | > 60% é bom |
| **Taxa Conversão** | % de leads que viraram clientes pagos | > 30% é excelente |

### Como Interpretar o Score

```
0-20%   → ❌ Não qualificado (falta informação)
20-50%  → ⚠️  Pouco viável (questionar mais)
50-70%  → ✅ Qualificado (bom pra vender)
70-100% → 🔥 Muito viável (vender URGENTE)
```

### Relatórios Semanais Recomendados

**Segunda-feira:**
- Leads gerados na semana anterior
- Conversão por tipo de benefício
- Performance de cada especialista

**Quinta-feira:**
- Leads de alto score (>80%)
- Leads em aberto (sem follow-up)
- Taxa de conversão acumulada

## ❓ FAQ

### **P: Como vejo um lead que chegou ontem?**
**R:** Menu → Filtros → Período "Últimas 24 horas" → Veja lista

### **P: Um cliente mudou de ideia. Como remarcar?**
**R:** Abra o lead → Veja histórico → Clique "Remarcar conversa" → Escolha horário

### **P: Quanto custa por lead qualificado?**
**R:** O sistema é mensal. Custa fixo. Quanto mais leads, melhor o ROI!

### **P: Posso integrar com meu sistema de CRM antigo?**
**R:** Sim! Menu → Configurações → Integrações → Conecte sua ferramenta

### **P: Os dados de clientes são confidenciais?**
**R:** 100% seguro. Criptografado. LGPD compliant. Apenas usuários autorizados acessam.

### **P: Como faço se o cliente prefere falar com humano?**
**R:** A IA roteia automaticamente se o cliente pedir. Menu → Equipe → Atendentes

### **P: Posso treinar a Marta com meu próprio roteiro?**
**R:** Sim! Menu → Configurações → Roteiros → Customize as 10 perguntas

### **P: Qual é o melhor horário para lead ligar?**
**R:** Menu → Analytics → Veja gráfico "Conversas por Hora" → Identifique pico

### **P: Como eu saibo se um lead está gerando vendas?**
**R:** Menu → Dashboard → Taxa de Conversão atualiza em tempo real

### **P: Preciso instalar algo no computador?**
**R:** Não! É 100% web. Funciona em qualquer navegador (Chrome, Safari, Firefox)
